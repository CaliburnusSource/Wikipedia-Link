software y otros tipos de obras.

cualquier otro trabajo publicado esta forma por sus autores. Puedes aplicarlo a

precio. Nuestras Licencias Públicas Generales están diseñadas para asegurarnos de que usted
tener la libertad de distribuir copias de software libre (y cobrar por

quiere, que puede modificar el software o usar fragmentos de él en nuevos
programas libres, y que usted sabe que puede hacer estas cosas.
